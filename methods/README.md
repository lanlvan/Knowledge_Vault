# Methods

* `methods` 存放跨项目复用的方法；
* 例如 PRD 交接、页面识别、AI + Codex 工作流、复盘方法等；
* 不存放具体项目事实；
* 具体项目事实仍放 `projects/`；
* 当前仅作为入口，后续按真实使用新增方法文件。

## 方法入口

* LLMWiki 知识层试运行方案：`llmwiki-knowledge-layer-trial.md`（记录 LLMWiki 知识层试运行方法）
* Obsidian 轻量维护方法：`obsidian-light-maintenance.md`（说明如何使用 Obsidian 对 Knowledge Vault 进行日常轻量维护、信息状态分层和人工更新）
