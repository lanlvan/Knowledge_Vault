# Personal Work

* `personal-work/` 用于承接非单一项目的个人工作沉淀；
* 承接范围包括：部门信息、周报 / 月报 / 日报、会议纪要、团队管理、绩效 / Owner 制、AI 工作流治理、个人阶段复盘、跨项目协作问题、向上汇报材料、管理判断；
* 不强行归入具体项目，如后续内容变成项目，可迁入 `projects/`。

## 不承接范围

* BOMCASE 页面事实源；
* BOMCASE 页面交接；
* BOMCASE 项目业务语境；
* BOMCASE 项目 `pending`；
* BOMCASE 项目 `decisions`；
* 具体项目 `source`。

## 当前最小结构

* `sources/`：非项目类个人工作原始材料；
* `briefs/`：非项目类个人工作摘要；
* `pending.md`：非项目类待确认问题；
* `decisions.md`：非项目类已形成判断。
* `information-map.md`：用于说明 personal-work 的信息分类、建议位置和 AI 读取路径。
