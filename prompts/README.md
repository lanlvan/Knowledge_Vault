# Prompts

* `prompts` 存放跨项目复用的 prompt；
* 项目专用 prompt 放在对应项目的 `prompts/` 目录；
* prompt 不承载业务事实；
* prompt 只描述执行方式和约束；
* 当前仅作为入口，后续按真实使用新增通用 prompt。
* `daily-report-from-vault.md`：用于基于 Knowledge Vault 生成日报、阶段汇报或轻量工作同步内容。
* `report-input-template.md`：用于在生成日报、周报、月报、阶段汇报或会议准备前，收集最小输入信息。
* `historical-material-intake.md`：用于将历史材料受控归位到 Knowledge Vault，判断归属、有效性、brief、pending、decisions 与 correction 处理方式。
