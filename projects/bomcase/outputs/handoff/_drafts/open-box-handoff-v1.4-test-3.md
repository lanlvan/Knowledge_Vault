## 2. 不变更范围声明

- 不改变开箱页三栏职责（左侧资源概览 / 中部主判断与主操作 / 右侧选卡填充）。
- 不改变保底模块展示条件与职责边界（仅配置保底玩法时展示；不参与满足度计算，不影响按钮主状态）。
- 不改变能量溢出提示展示位置（仅右侧卡牌背包全展开态底部固定汇总区）。
- 不改变 X1/X5 切换后“已选清空 + 状态重算”的页面行为。
- 不改变既有验收基线与已有功能边界（资源扣减优先级、完整能量计算、奖励结算等仍不在本交付内重定义）。

## 3. 研发影响标记

### 3.1 【本次延续】高影响模块最小实现口径：能量溢出提示

- 模块职责：在右侧卡牌背包全展开态底部固定汇总区，辅助解释“本次已选卡牌能量超出当前开启所需补足能量”后的沉淀结果。
- 已确认展示口径：
  - 仅当存在溢出时展示“溢出 +Y% 存入能量余额”；
  - 无溢出时不展示该提示；
  - 该提示不进入中部满足度区、资源构成区或左侧资源概览。
- 关键状态 / 交互：
  - 已选卡牌变化会触发底部汇总重算；
  - 切换 X1/X5、清除已选后，溢出提示同步更新或清空；
  - 半展开态不展示溢出提示。
- 研发影响侧：前后端。
- 待确认阻塞点：
  - 能量溢出值来源（后端返回或前端可计算）；
  - 能量溢出值刷新时机（选卡变化、X1/X5 切换、清除已选、开启后）。
- 验收关注点：
  - 展示条件与隐藏条件正确；
  - 展示区域固定在右侧全展开底部；
  - 与中部/左侧展示口径不混用。

### 3.2 【未变更，沿用上一版口径】其它研发影响聚焦

- 保底模块数据与结果态以后端配置和返回为准。
- 批量开启跨轮结果由后端结算给出，前端不自行推导优先级。

## 4. 页面目标【未变更，沿用上一版口径】

- 左侧展示资源总览；
- 中部承接开启判断与主操作；
- 右侧承接卡牌填充选择；
- 中部展示保底进度与规则说明入口；
- 下方饰品区承接保底范围标记关联表达。

## 5. 页面结构【未变更，沿用上一版口径】

1. 顶部导航（已有功能范围）。
2. 左侧资源概览区。
3. 中部猎户军需核心区（X1/X5、满足度、资源构成、CTA）。
4. 中部保底进度模块（条件展示）。
5. 右侧卡牌背包区（半展开摘要态 / 全展开操作态 / 全展开底部固定汇总区）。
6. 下方饰品区保底标记与底部掉落信息 Tab（已有功能边界）。

## 6. 组件与字段

### 6.1 【本次调整】高关注字段索引（优先联调）

- 右侧底部汇总：`已选 N 张｜填充 +X%｜清除已选`
- 右侧溢出提示：`已选卡牌将在开启时消耗，溢出 +Y% 存入能量余额`
- 中部保底文案：`当前 X/Y，再积累 N 次可获「保底奖励名称」`

### 6.2 【未变更，沿用上一版口径】其余组件字段

- 左侧资源字段（赠品次数、能量余额、机械扳手、卡牌入口、会员等级）保持既有口径。
- 中部资源构成 icon 区仍仅展示本次实际参与资源。
- 右侧顶部不展示未确认口径计数（如 `[0/3]`、`3/300`）。

## 7. 核心页面口径

### 7.1 【未变更，沿用上一版口径】三栏职责口径

- 左侧只做资源概览，不承担开启判断；
- 中部是主判断与主操作区；
- 右侧是选卡填充选择器，不是资产管理区。

### 7.2 【本次延续】能量溢出口径（高关注）

- 溢出提示属于页面展示口径补充，不构成资源计算规则变更。
- 展示位置限定在右侧全展开底部固定汇总区。
- 不改变中部主操作区职责与开启按钮主状态。

### 7.3 【未变更，沿用上一版口径】保底模块口径

- 仅配置保底玩法时展示，未配置不占位；
- 不参与开启满足度计算，不进入资源构成 icon 区；
- 保底相关数据与结果以后端返回为准。

## 8. 页面状态与交互规则

### 8.1 【未变更，沿用上一版口径】关键状态

- 半展开与全展开共用同一组件骨架；
- 一键填充跳过喜欢卡时展示轻提示；
- 开启条件不足时，开启军需按钮仍可点击并触发已有能力承接。

### 8.2 【本次延续】溢出相关状态联动

- 有溢出：底部固定汇总区展示溢出提示；
- 无溢出：仅展示“已选卡牌将在开启时消耗”；
- 清除已选 / 切换 X1X5：溢出提示与已选状态同步清空。

## 9. 空状态 / 异常状态【未变更，沿用上一版口径】

- 无可用卡牌：右侧半展开显示空态插图 + “前往获取”入口；
- 保底模块空状态：未配置保底玩法时不展示、不占位；
- 溢出提示空状态：不存在溢出时不展示。

## 11. 验收口径

### 11.1 【本次调整】优先验收项

- 存在溢出时：右侧全展开底部显示溢出提示。
- 不存在溢出时：不显示溢出提示。
- 溢出提示不出现在中部满足度、资源构成 icon 区或左侧资源概览。
- 切换 X1/X5 或清除已选后，溢出提示状态与当前选择一致。

### 11.2 【未变更，沿用上一版口径】结构与行为验收
