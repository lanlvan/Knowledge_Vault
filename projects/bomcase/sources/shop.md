## 2. Page Goal
<!-- section_layer: F+E -->
<!-- layer_note: 页面用途、用户可执行动作和详情/下单流程承接为 F；用户可见入口和按钮文案仅作辅助读取，正文文案以 Data / Fields 与 Interactions 章节为主，读取时必须局部拆分。 -->

该页面用于展示商城商品列表，并展示对应的军需权益内容。

用户可在当前页面：

- 查看商品信息；
    
- 查看商品对应的军需权益；
    
- 切换权益选中项；
    
- 通过商品图片或标题进入商品详情页；
    
- 点击“立即下单”进入已有下单流程。
