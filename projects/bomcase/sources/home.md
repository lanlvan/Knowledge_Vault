# Home Active Source

## 1. Source Scope

本文件是 Home 页面当前 active source（`source_type: handoff_document`），用于维护页面事实、字段、状态、交互与验收口径。

事实冲突时以本文件为准；本文件不替代完整系统规则库。

## 2. Page Goal

当前页面用于展示：

- 商城首页 Banner 活动内容；
    
- 热门掉落展示；
    
- 军需权益展示；
    
- 商品列表；
    
- 商品赠品信息；
    
- 玩法流程说明；
    
- 支付后说明；
    
- 平台信任背书信息。
    

支持用户：

- 浏览 Banner 活动；
    
- 查看热门掉落；
    
- 查看可获军需权益；
    
- 进入商品详情；
    
- 选择赠品；
    
- 进入下单流程；
    
- 查看概率公示。

## 6. Interactions

|触发对象|触发条件|点击结果|适用范围|不适用处理|
